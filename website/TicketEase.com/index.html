FAQs

Get quick answers to the most
popular questions
I don't know how many people will show up directly at the event. How can I use Ticket Generator?
You can use Ticket Generator to generate and download ticket images/PDFs. You can print these tickets and sell them directly at the box office. Alternatively, you can use the web application in real-time to generate and send the tickets one-by-one at the box office.
I already have a ticket design. How can Ticket Generator help?
Using Ticket Generator, you can upload your own ticket design (say created using Canva or Adobe Spark) while ticket generation. Then, Ticket Generator will help you add a unique QR Code, ticket code, and unique information such as guest name, hall number, and seat number. Each ticket with your design and Ticket Generator's security can now be validated by event coordinators at the entry gate using the Ticket Validator mobile app.
I don't even have a ticket design. How can Ticket Generator help?
If you're starting from scratch, you can use Ticket Generator's standard template to quickly generate tickets. You only need to provide event details (name, description, venue, date, and timings) and branding (logo, background color/image). You can also add unique ticket information to each ticket such as guest name, hall number, seat number, or any other detail as per your needs. Each ticket will have a QR Code that can be validated at the entry using the Ticket Validator app.
I already have an events app to get registrations and payments. How can I use Ticket Generator?
If you have an event registration website to allow attendees register for your event and purchase tickets, you can use Ticket Generator API. It will help you programmatically generate secure tickets in real-time. As soon as ticket purchase is successful, make a POST call to Ticket Generator API and get a QR Code ticket that can be shown in the app or sent directly to the attendee.
I already have the guest list. Can I directly send tickets without any events page or registrations?
Yes. If you already have the guest list with their contact information (email/phone number), you can upload this information in the application and directly send tickets. You can send tickets via email, SMS, and WhatsApp. You can also download the ticket images/PDFs for printing, if required.
I'm organizing an event. What can I use Ticket Generator for?
Say you're organizing an event for 20 people or even upto 10,000 people. You can either use a ticketing platform that charges exorbitant commissions or you can directly use Ticket Generator. It is an easy-to-use online tool that you can use to create an event page, get registrations/payments, approve registrations, design tickets, download/send tickets, validate tickets at entry, and get attendance data. Compared to standard ticketing platforms, Ticket Generator is highly affordable, helping you increase your event margins.
